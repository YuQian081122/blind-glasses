韌體指令面板（PowerShell）

依序執行即可。每個區塊都可單獨複製。適用專案：blind_glasses / firmware。

Step 1

切到 firmware 目錄

確保所有 PlatformIO 指令在正確路徑執行。

cd C:\Users\sl131\Downloads\blind_glasses\firmware
已複製
Step 2

編譯韌體

先確認程式可正常編譯，再進行燒錄。

cd C:\Users\sl131\Downloads\blind_glasses\firmware
pio run
已複製
Step 3

燒錄到 ESP32-S3

自動偵測 COM port 並上傳韌體。

cd C:\Users\sl131\Downloads\blind_glasses\firmware
pio run -t upload
已複製
Step 4

開啟 Serial Monitor

監看 IMU 診斷輸出（115200）。

cd C:\Users\sl131\Downloads\blind_glasses\firmware
pio device monitor -b 115200
已複製
IMU Test

啟用 IMU 單獨測試（目前已開）

目前 `include/config.h` 內 `IMU_STANDALONE_TEST=1`，可直接燒錄測試。

cd C:\Users\sl131\Downloads\blind_glasses\firmware
pio run -t upload
pio device monitor -b 115200
已複製
Optional

回復一般模式

把 `IMU_STANDALONE_TEST` 改回 `0` 後，重新燒錄。

cd C:\Users\sl131\Downloads\blind_glasses\firmware
pio run -t upload
已複製
Optional

清理建置快取

遇到奇怪編譯問題時可先 clean 再 build。

cd C:\Users\sl131\Downloads\blind_glasses\firmware
pio run -t clean
pio run
已複製
